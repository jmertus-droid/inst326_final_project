# CareerPath Application Tracker

## Project Description

CareerPath Application Tracker is a Python program that helps users organize job and internship applications. The program allows users to add applications, view saved applications, filter applications by status, update application statuses, delete applications, and calculate a priority score for each application.

This project was created for INST326: Object-Oriented Programming for Information Science. It demonstrates object-oriented programming, user input validation, file handling, JSON storage, control flow, functions, and testing.

## Features

- Add a new job or internship application
- View all saved applications
- Filter applications by application status
- Update the status of an existing application
- Delete an application
- Calculate a priority score based on:
  - Interest level
  - Skill match
  - Deadline urgency
- Identify missing skills for each application
- Save and load application data using a JSON file
- Handle invalid user input
- Include test cases using Python's `unittest` module

## Files Included

- `main.py`  
  Main program file containing the application tracker logic.

- `test_main.py`  
  Test script used to test important program features.

- `applications.json`  
  JSON file used to store saved application data.
